# Console Shooting Game (C#)

간단한 콘솔 환경에서 동작하는 2D 슈팅 게임입니다.
플레이어를 이동시키며 적을 제거하는 구조로, 키보드 입력과 콘솔 좌표를 활용한 기본 게임 루프를 구현했습니다.

플레이어를 움직이며 화면 상단에 있는 모든 적을 제거하면 게임이 종료됩니다.

---

## 개요

* **플랫폼**: C# Console Application
* **개발 환경**: .NET (C#)
* **입력 방식**: 키보드 (WinAPI `user32.dll`)
* **렌더링 방식**: 콘솔 좌표 출력 (`Console.SetCursorPosition`)

---

## 주요 기능

### 1. 플레이어 이동

* 방향키 입력을 통해 플레이어 이동
* 벽을 벗어나지 않도록 제한 처리 (`HitWall()`)

### 2. 총알 발사

* Space 입력 시 총알 생성
* 총알은 위쪽으로 이동
* 이동 중 적과 충돌 시 데미지 적용

### 3. 적 시스템

* 일정 좌표에 적 생성
* 체력(Hp)이 0이 되면 제거

### 4. 충돌 처리

* 총알 위치와 적 위치 비교
* 적 범위(`Enemy.MOB.Length`) 내 충돌 판정

### 5. 화면 렌더링

* 매 프레임 `Console.Clear()` 후 전체 다시 출력
* 벽, 적, 플레이어 순서로 렌더링

---

## 클래스

### Player

* 플레이어 위치 관리
* 이동 및 발사 처리
* 총알 리스트 관리

### Enemy

* 적 위치, 체력, 생존 여부 관리
* 콘솔에 적 출력

### Bullet

* 총알 위치 및 이동 처리
* 적과 충돌 시 데미지 계산

### Wall

* 게임 영역 테두리 출력
